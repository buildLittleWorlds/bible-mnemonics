## How to Use This Queue

1. **Pick a passage** — Start with Tier 1 or whatever draws you
2. **Tell Claude** — "Let's do Romans 5:1"
3. **Generate image(s)** — Claude selects ore source, writes prompt, you generate
4. **Assign loci** — Map phrasal units to objects in the image
5. **Update status** — Change from "queued" to "completed" in JSON
6. **Memorize** — Walk the image, practice recall
